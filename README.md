# mind-maze
ESP/Arduino IOT project for dementia and alzheimer's risk stratification; Final Coursework for CM3040


Libraries Used: 

| Library Name | Version | Used In Module | Purpose |
| :--- | :--- | :--- | :--- |
| **Adafruit GFX Library** | `1.11.x` | Diagnostic | Core graphics primitives for the Sharp Memory Display. |
| **Adafruit Sharp Memory Display** | `1.1.x` | Diagnostic | Hardware driver for the Sharp Memory LCD |
| **ArduinoJson** | `6.21.x` | Both | Serialization of JSON data for backend communication. |
| **LiquidCrystal** | `1.0.x` | Monitoring | Driver for the 2x16 Character LCD. |
| **WiFiS3** | `Built-in` | Both | Network connectivity for the Arduino Uno R4 WiFi. |
